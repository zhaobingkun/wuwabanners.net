How the site is structured

The site now aims to follow one simple pattern: navigation first, list page second, detail page third. That keeps the structure easier to scan on both mobile and desktop.

1. Start from navigation

Use the top navigation to choose the branch: banners, guides, characters, weapons, or items.

2. Open a list page

Each branch should give you a clean list layer first, such as a hub, directory, timeline list, or reference gallery.

3. Open a detail page

From there, move into a narrower page for one banner phase, one character, one weapon, or one item without hitting dead ends.
