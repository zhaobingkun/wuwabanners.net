# Search Console Submit Order

Use this order after a deployment or a major banner-cycle update.

## Rule

Do not request indexing for every page at once.

Start with the pages that:

- define the site structure
- receive the strongest internal links
- answer the broadest search intents

Then move down into support pages and character pages.

## Batch 1: Site entry points

Submit these first:

1. `https://wuwabanners.net/`
2. `https://wuwabanners.net/banners/`
3. `https://wuwabanners.net/guides/`
4. `https://wuwabanners.net/sitemap.xml`

Why:

- They define the site structure.
- They help Google discover the rest of the site faster.

## Batch 2: Core money pages

Submit these next:

1. `https://wuwabanners.net/wuthering-waves-next-banner/`
2. `https://wuwabanners.net/wuthering-waves-current-banner/`
3. `https://wuwabanners.net/pull-advice/`
4. `https://wuwabanners.net/wuthering-waves-banner-history/`

Why:

- These are the strongest intent pages right now.
- They are the most likely to attract early impressions.

## Batch 3: Banner support cluster

Submit these after Batch 2:

2. `https://wuwabanners.net/wuthering-waves-next-rerun/`
3. `https://wuwabanners.net/wuthering-waves-pity-system/`
4. `https://wuwabanners.net/wuthering-waves-banner-schedule/`
5. `https://wuwabanners.net/wuthering-waves-weapon-banner/`
6. `https://wuwabanners.net/wuthering-waves-all-banners/`
7. `https://wuwabanners.net/wuthering-waves-next-character/`
8. `https://wuwabanners.net/wuthering-waves-timeline/`
9. `https://wuwabanners.net/wuthering-waves-daily-reset-time/`
10. `https://wuwabanners.net/wuthering-waves-weekly-reset-time/`
11. `https://wuwabanners.net/wuthering-waves-event-calendar/`
12. `https://wuwabanners.net/wuthering-waves-banner-order/`

Why:

- These pages support the main banner pages.
- They help the site look like a content cluster instead of a few isolated pages.

## Batch 4: Character decision pages

2. `https://wuwabanners.net/wuthering-waves-should-you-pull-mornye/`

5. `https://wuwabanners.net/wuthering-waves-should-you-pull-chisa/`
6. `https://wuwabanners.net/wuthering-waves-should-you-pull-phrolova/`

Why:

- These are valuable, but more specific.
- They usually benefit from Google already understanding the site-level banner cluster first.

## After each future phase change

When the current and next banners change, request reindexing in this order:

1. `/wuthering-waves-next-banner/`
2. `/wuthering-waves-current-banner/`
3. `/wuthering-waves-banner-countdown/`
4. `/pull-advice/`
5. any regenerated `should-you-pull-*` pages for the new phase

## Practical note

You do not need to submit every page every day.

For ordinary updates:

- resubmit the sitemap
- request indexing for the handful of pages that changed most

That is usually enough.
