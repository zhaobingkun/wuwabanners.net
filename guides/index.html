Wuthering Waves Guides Hub

Use the guides branch when the user already understands the banner names and now needs a planning answer. The clean route is guides hub, then one decision page, then one character-specific support page if the question gets narrower.

Direct answer: Start here for pity, pull-or-save decisions, rerun judgment, and the first layer of character support. Open the decision page first, then move into character materials, build, or team comps only when the user needs follow-up depth.

How to use the guides branch

1. Start from the hub

Use the hub when the user needs a decision page, not just a lineup page. This is where pity, saving, reruns, and support-page routing belong.

2. Open one planning page

Move into pity, pull advice, or rerun first. That keeps the main decision answer visible before the user drops into narrower character pages.

3. Go into support pages last

Only after the main decision is clear should the user move into one character support page such as materials, build, or team comps.
